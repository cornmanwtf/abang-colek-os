## 1. Brand Overview

### 1.1 Brand Story

**Abang Colek** adalah jenama sambal/colek premium dari Malaysia yang membawa rasa pedas autentik ke seluruh Asia Tenggara. Bermula dari dapur rumah, kini Abang Colek telah menjadi pilihan utama peminat pedas di bazaar, festival makanan, dan platform online.

### 1.2 Brand Values

| Value | Description |
| ----- | ----------- |
| **Authenticity** | Rasa asli, resepi turun-temurun |
| **Bold** | Berani pedas, berani beza |
| **Community** | Membina komuniti penggemar pedas |
| **Quality** | Bahan berkualiti, tiada kompromi |

### 1.3 Brand Personality

- **Playful** - Fun, tidak terlalu serius
- **Street Culture** - Urban, hypebeast vibes
- **Local Pride** - Bangga dengan rasa tempatan
- **Confident** - Yakin dengan kualiti produk

### 1.4 Target Audience

| Segment | Demographics | Psychographics |
| ------- | ------------ | -------------- |
| **Primary** | 18-35 tahun | Foodie, suka cuba benda baru |
| **Secondary** | 35-50 tahun | Ibu bapa, cari sambal berkualiti |
| **Tertiary** | Students | Budget-conscious, suka viral food |

### 1.5 Brand Tagline

**Primary:** *"Rasa Padu, Pedas Menggamit"*

**Secondary Options:**

- "Pedas Tapi Puas"
- "Colek Sampai Licin"
- "Bukan Calang-Calang Pedas"

| ----- | ------ | --- |

| Regular | 400 | Subheadings |

### 5.2 Secondary Font

**Font Family:** Poppins / Inter (for body)

| Style | Weight | Use |
| ----- | ------ | --- |
| Bold | 600 | Emphasis |
| Medium | 500 | Subheadings |
| Regular | 400 | Body text |
| Light | 300 | Captions |

### 5.3 Type Scale

| Level | Size | Line Height | Use |
| ----- | ---- | ----------- | --- |
| H1 | 48px | 1.2 | Main headlines |
| H2 | 36px | 1.3 | Section titles |
| H3 | 24px | 1.4 | Subheadings |
| Body | 16px | 1.5 | Paragraphs |
| Caption | 12px | 1.4 | Small text |

### 5.4 Text Styling

**Headlines:**

- ALL CAPS for impact
- Letter spacing: 2-4%
- Color: Sambal Red or Midnight Black

**Body:**

- Sentence case
- Max line width: 70 characters
- Color: Midnight Black

## 12. Do's and Don'ts

### 12.1 Logo Don'ts

| âŒ Don't | Why |
| -------- | --- |
| Stretch or compress | Distorts proportions |
| Change colors | Breaks brand consistency |
| Add effects (shadow, glow) | Looks unprofessional |
| Place on busy backgrounds | Reduces visibility |
| Outline the logo | Alters appearance |
| Rotate | Disrupts recognition |

### 12.2 Color Don'ts

| âŒ Don't | Why |
| -------- | --- |
| Use pink/purple | Not brand colors |
| Low contrast combinations | Hard to read |
| Neon versions | Looks cheap |
| Gradient overuse | Distracting |

### 12.3 General Don'ts

| âŒ Don't | Why |
| -------- | --- |

| Political content | Alienates audience |

## 14. Motion Graphics & Animation

### 14.1 Animation Style

| Type | Style |
| ---- | ----- |
| **Transitions** | Smooth, bouncy (ease-out) |
| **Text Animation** | Pop-in, slide-up |
| **Logo Animation** | Chili bounce, leaves flutter |
| **Mascot** | Subtle movements, blink, wave |

### 14.2 Video Intro/Outro

**Intro (3 seconds):**

1. Yellow background fade in
2. Logo pop with bounce effect
3. Chili sparkle SFX

**Outro (3 seconds):**

1. Content shrinks
2. Logo + TikTok handle appear
3. "Follow for more!" text

### 14.3 Motion Specifications

| Parameter | Value |
| --------- | ----- |
| Frame Rate | 30fps (social), 60fps (premium) |
| Duration (logo reveal) | 2-3 seconds |
| Ease Type | Cubic-bezier(0.34, 1.56, 0.64, 1) |
| Export Format | MP4 (social), MOV (editable) |

### 14.4 GIF/Sticker Guidelines

- Max 3 seconds loop
- Transparent background preferred
- Mascot expressions: excited, hungry, shocked
- Product animations: steam rising, chili bouncing
