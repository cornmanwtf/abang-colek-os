## Overview

A modern, digital-first lucky draw system designed for Abang Colek's 2026 events. This system replaces traditional paper-based entry with QR code registration and viral TikTok integration.

**Key Features:**

- Zero paper waste (digital registration)
- Built-in viral marketing (TikTok requirement)
- Automatic database collection
- Transparent live draw system
- Upselling framework for staff

## 5. Upselling Strategy

### 5.1 Staff Scripts

**Scenario 1: Customer wants 1-2 units**
> "Bang, tambah sikit je cukupkan 5. Boleh join Lucky Draw iPhone 17! Tahun ni hadiah gempak."

**Scenario 2: Customer at 5 units**
> "Alang-alang 10 terus jadi VVIP! Peluang menang double. Dapat 2 tiket terus."

**Scenario 3: Customer says "mahal"**
> "Bukan mahal bang, tu pelaburan. RM75 je dapat chance menang iPhone RM7,000. Berbaloi!"

**Scenario 4: Customer hesitates**
> "Ramai dah join tadi. Tiket tinggal sikit je. Last day esok tau."

### 5.2 Psychology Techniques

| Technique | Example | Effect |
| --------- | ------- | ------ |
| **FOMO** | "Ramai dah join" | Fear of missing out |
| **Scarcity** | "Stok tinggal 20" | Urgency |
| **Social Proof** | "Tadi family ambil 10" | Follow the crowd |
| **Anchoring** | "iPhone 17 worth $1500" | Value comparison |

## 6. Draw Execution

### 6.1 Setup Requirements

| Item | Purpose |
| ---- | ------- |
| Laptop | Run randomizer software |
| TV/Monitor | Display to crowd |
| Phone | TikTok/IG Live + Video call winner |
| Internet | Stable connection |

### 6.2 Software Options

- [Wheel of Names](https://wheelofnames.com/) - Free, visual
- [Random Name Picker](https://miniwebtool.com/random-name-picker/) - Simple
- Google Sheets `=RANDBETWEEN()` - DIY option

### 6.3 Draw Protocol

1. **Pre-Draw (11:30 PM)**
   - Export all entries from database
   - Load into randomizer
   - Test run (don't save)

2. **Live Draw (12:00 AM)**
   - Start TikTok/IG Live
   - Show all names on screen (transparency)
   - Build suspense with countdown
   - Spin/pick winner

3. **Winner Verification**
   - Video call winner immediately
   - Verify dengan No. Phone yang didaftar
   - Confirm identity dengan IC
   - Announce on Live

4. **Post-Draw**
   - Screenshot/record everything
   - Post winner announcement
   - Arrange prize collection

## 10. FAQ

**Q: Kalau pelanggan tak nak post TikTok?**
A: Tiket still valid, tapi tak dapat instant reward. Galakkan mereka post untuk hadiah kecil.

**Q: Kalau winner tak angkat call?**
A: Cuba 3 kali. Kalau tak jawab, cabut nama kedua sebagai backup.

**Q: Macam mana verify pemenang?**
A: Check No. Phone dalam database. Winner mesti guna phone yang sama untuk jawab video call.

**Q: Boleh ke satu family share tiket?**
A: Tidak. Setiap entry daftar dengan satu phone number sahaja.
