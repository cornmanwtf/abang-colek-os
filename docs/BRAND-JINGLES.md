## 4. TikTok Hook Versions (5s)

### 4.1 Version Analysis

Each hook serves a different content strategy:

| Version | Style | Best For | Virality Factor |
| ------- | ----- | -------- | --------------- |
| **A** | Classic chant | Brand intro/outro | High recognition |
| **B** | Challenge hook | User participation | High engagement |
| **C** | Call-response | Duet content | High duet rate |
| **D** | Question hook | Comment bait | High comments |

### 4.2 Hook A - Classic Chant

```text
A-BANG CO-LEK! 
Pedas tapi puas! üî•
```

**Rationale:**

- 2 syllables per beat (A-BANG, CO-LEK)
- Contrast statement (pedas BUT puas)
- Emoji-friendly ending for captions

### 4.3 Hook B - Challenge Hook

```text
Colek-colek-colek!
Siapa boleh tahan?! 
*beat drop*
```

**Rationale:**

- Triple repetition = memorable
- Question format invites response
- Beat drop marks transition moment

### 4.4 Hook C - Call-Response

```text
ABANG! (Colek!)
ABANG! (Colek!)
Rasa PADU! Pedas GILA! 
```

**Rationale:**

- Perfect for duet split
- Builds anticipation
- Caps lock emphasis on key words


## 8. Lyric Writing Guidelines

### 8.1 Language Rules

| Rule | Guideline | Example |
| ---- | --------- | ------- |
| **Primary** | Malay (Bahasa Malaysia) | "Rasa padu" |
| **Secondary** | English mix | "keeping it real" |
| **Slang** | Street/youth slang | "gempak", "legit" |
| **Avoid** | Formal Malay | "sedap sekali" ‚Üí "sedap gila" |

### 8.2 Brand Words Dictionary

**Always Use:**

| Word | Meaning | Context |
| ---- | ------- | ------- |
| "Padu" | Solid/excellent | Quality descriptor |
| "Gila" | Crazy (positive) | Intensity |
| "Gempak" | Awesome | Praise |
| "Legit" | Legitimate | Authenticity |
| "Colek" | Dip | Product action |

**Never Use:**

| Word | Reason |
| ---- | ------ |
| Competitor names | Legal issues |
| Negative spice words | Brand damage |
| Overly formal language | Off-brand |

### 8.3 Rhyme Schemes

| Pattern | Example | Use Case |
| ------- | ------- | -------- |
| AABB | gila/lagi | Chorus |
| ABAB | pack/grand/belakang/champion | Verses |
| Chant | Repetition | Hooks |

### 8.4 Syllable Counts

| Section | Target | Reason |
| ------- | ------ | ------ |
| Hook | 4-8 syllables | Memorable |
| Chorus line | 8-12 syllables | Singable |
| Verse line | 12-16 syllables | Flow |
