# STACK_PROFILES

## Profile A: Modern Web SaaS (Balanced)
- Backend: Node.js (NestJS) or Go (Gin)
- Frontend: React (Vite) + Tailwind
- Data: PostgreSQL + Redis
- Cloud: AWS (ECS/Fargate) + RDS + S3
- Observability: OpenTelemetry + Grafana

## Profile B: Enterprise Java
- Backend: Java (Spring Boot)
- Frontend: React (Vite)
- Data: PostgreSQL
- Cloud: AWS (EKS) + RDS + S3
- Observability: Prometheus + Grafana + Loki

## Profile C: Serverless First
- Backend: AWS Lambda (Node.js/TypeScript)
- Frontend: Next.js
- Data: DynamoDB
- Cloud: AWS (API Gateway, Step Functions)
- Observability: CloudWatch + X-Ray
